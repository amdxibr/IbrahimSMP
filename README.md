# History
This SMP was created by me during early May 2026, and has survived ever since. To ensure the safety of the SMP, I've also decided to back it up here on GitHub.
It consists of my home at spawn which i call BEIT IBRAHIM, a castle of my friend whose name is immadthechad (The castle's name is Immadthechad Castle, and is located 700 blocks to the North of Beit Ibrahim), Beit Abdullah wa Ausaf (for my two friends)...
These landmarks aren't just arbitrarily scattered... They're connected by a well-developed system of well-lit underground tunnels.
